Usability Results

Lack of Navigation

User expressed the need for more navigation on the final page of the project in order to got back and view other income ranges easily.

I plan to add some sort of return button to allow the user to see the other income ranges. I also envisioned the ability to change the highlighted sections from the final page to remove an extra step and unnecessary loading. Additional styling to the already existing navigation is necessary for the overall direction of the project.

Directions Simple & Clear

User expressed liking of the simple directions and statistics provided.

This was a successful part of the project that I planned for. I wanted the user to be able to easily navigate and understand the meaning of the project. I believe that the project require more narrative to drive the point through, but I will keep it as minimal and simple as possible.

Hierarchy of Data

I wanted to ask which part of the information provided on each state the user thinks is most important. User expressed importance for COL/wage.

I want the user to be immediately be able to read the information provided for exactly what they need when they choose a state. I need to reduce the information provided down and prioritize the most important information.
