Link Display Function into Arrow Function

This function is used in my check yourself mini-studio. I used it to change the HTML on the page according to which link the user chooses.

Link Display Function

            function outputHTML(data){
                const display = document.querySelector("#display")
                display.innerHTML = 
                `page filler`;
            }

Function Expression

            const outputHTML = function(data){
                const display = document.querySelector("#display")
                display.innerHTML = 
                `page filler`;
            }

Arrow Function

            const outputHTML = (data => {                
                const display = document.querySelector("#display")
                display.innerHTML = 
                `page filler`;
            });
